# Audimat
a rack for hosting audio plugins

named after the Automat - a famous self-service eatery in New York

<a href="https://kohoutech.github.io/Audimat/">project documentation</a>

it requires the following library packages, which can be installed via NuGet  
<a href="https://www.nuget.org/packages/Kohoutech.ENAML/">ENAML</a>  
<a href="https://www.nuget.org/packages/Kohoutech.Midi/">Midi</a>  
<a href="https://www.nuget.org/packages/Kohoutech.Patch/">Patch</a>  
<a href="https://www.nuget.org/packages/Kohoutech.VST/">VST</a>  
<a href="https://www.nuget.org/packages/Kohoutech.Widget/">Widget</a>
